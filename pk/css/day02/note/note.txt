9》伪类选择器
LVHA的顺序去记

:hover->鼠标悬浮到元素上的样式

:focus->具有输入域的元素设置的样式

10》伪元素
:after->在当前元素的内容后添加内容
:before->在当前元素的内容前添加内容

	content:"内容";
}
:before{
	content:url(目标文档的URL);
}

1.浮动:

float:left|right

当子元素浮动的时候，会对父元素高度造成影响，高度会清空为0
对父元素应用样式：
.clear{
	overflow:hidden|auto;
	zoom:1;
}


清除浮动：clear:left|right|both;
clear:both,清除浮动对下级造成的影响
